# **\[디자인 정의서\] 로컬 지식 봇 (Local-First UI/UX)**

## **1\. 디자인 컨셉 (Concept)**

* **Keyword**: "Professional, Trustworthy, Simple"  
* **컨셉**: 사내 규정을 다루는 만큼 신뢰감을 주는 '딥 블루'와 '그레이' 톤을 기반으로 하되, 대화의 즐거움을 주는 챗봇 레이아웃 채택.

## **2\. 컬러 팔레트 (Color Palette)**

* **Primary**: \#1A73E8 (신뢰감을 주는 코포레이트 블루)  
* **Background**: \#F8F9FA (눈이 편안한 밝은 그레이)  
* **Surface**: \#FFFFFF (깨끗한 화이트 카드 스타일)  
* **Text**: \#202124 (가독성이 높은 다크 차콜)

## **3\. 화면 구성 (Page Layout)**

전체 화면을 크게 2개의 영역으로 나눈 **'사이드바 \+ 채팅창'** 구조를 제안합니다.

### **A. 사이드바 (Local Document Monitor) \- 좌측 (25%)**

* **상단**: 로고 및 "로컬 지식 봇" 타이틀.  
* **중단**: '현재 학습된 문서' 리스트 (내 PC 폴더 내 20\~50개 파일명 표시).  
* **하단**: 보안 상태 표시기 ("Local Mode Only" 녹색 아이콘) 및 API 연결 상태.

### **B. 메인 채팅창 (Conversation Area) \- 우측 (75%)**

* **메시지 리스트**: 사용자와 AI의 대화가 카드 형태로 표시.  
  * 사용자: 우측 정렬, 블루 배경.  
  * AI: 좌측 정렬, 화이트 배경, 출처 표시 포함.  
* **채팅 입력창**: 하단 고정형 플로팅 박스.  
  * 질문 입력칸 (Textarea) \+ 전송 버튼 (Icon).  
  * "무엇이든 물어보세요 (예: 경조사 휴가 며칠이야?)" 플레이스홀더.

## **4\. 주요 UX 인터렉션 (Key Interactions)**

* **답변 생성 애니메이션**: 답변이 나올 때 타이핑 효과(Streaming)를 주어 실시간 소통 느낌 강조.  
* **출처 하이라이트**: 답변 하단에 "참고: \[파일명.pdf\]" 클릭 시 해당 파일이 어떤 내용인지 툴팁으로 간단히 보여줌.  
* **반응형 디자인**: PC 웹 브라우저 최적화 (창 크기에 따라 유연하게 조절).

## **5\. 특화 컴포넌트 (Special Components)**

* **Source Bubble**: 답변의 근거가 된 문서 조각을 별도 디자인된 작은 박스로 표시하여 정보의 신뢰도 확보.  
* **Copy Button**: AI 답변을 업무 메신저나 메일에 바로 복사해 붙여넣을 수 있는 간편 복사 버튼.
