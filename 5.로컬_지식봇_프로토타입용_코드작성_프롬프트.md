# **\[프로토타입 생성용 황금 프롬프트\]**

**아래 내용을 복사하여 AI(Gemini, ChatGPT 등)에게 입력하면 코드를 생성해 줍니다.**

### **\[Prompt\] 로컬 보안 지식 봇 개발**

**역할**: 너는 Node.js와 LangChain.js 전문가야. 사내 보안을 최우선으로 하는 '로컬 지식 봇'의 프로토타입 코드를 작성해줘.

**기능 요구사항**:

1. **로컬 RAG 시스템**: /documents 폴더의 PDF 파일을 읽어 HNSWLib 로컬 벡터 스토어에 저장하고 검색하는 로직을 구현해.  
2. **보안 지침**:  
   * 모든 API Key는 .env 파일의 GOOGLE\_API\_KEY를 참조해.  
   * 문서는 절대 외부 클라우드로 전송하지 말고 로컬에서만 텍스트를 추출해.  
   * AI에게는 검색된 관련 문서 조각(Context)만 전달해.  
3. **백엔드 (server.js)**:  
   * Express를 사용하여 /api/ask (질문 답변), /api/ingest (문서 학습) 엔드포인트를 만들어.  
   * 답변 시 반드시 하단에 "참고: \[파일명\]" 형태의 출처를 포함하도록 시스템 프롬프트를 설정해.  
4. **프론트엔드 (index.html)**:  
   * Tailwind CSS CDN을 사용하여 깔끔하고 모던한 채팅 UI를 만들어.  
   * 사이드바에는 현재 학습된 파일 목록이 보이고, 메인에는 대화창이 있어야 해.  
   * 답변은 실시간 느낌이 나도록 타이핑 효과를 구현해줘.

**기술 스택**:

* Node.js, Express, LangChain.js, @langchain/google-genai, HNSWLib-node, Tailwind CSS.

**제약사항**:

* 코드는 가독성이 좋게 주석을 상세히 달아줘.  
* 실행 방법(npm install 목록 등)을 포함해줘.
