# **\[기술 스택 및 개발 환경 정의서\] 로컬 지식 봇 (Antigravity Style)**

## **1\. 개발 철학 (Antigravity Principle)**

* **Zero Gravity**: 무거운 프레임워크를 배제하고, 최소한의 라이브러리로 최대의 성능을 낸다.  
* **Local First**: 모든 데이터 처리는 내 PC에서, AI 추론만 구글의 뇌(Gemini)를 빌려온다.  
* **Speed to PoC**: 셋업부터 첫 질문까지 30분 이내에 완료하는 것을 목표로 한다.

## **2\. 개발 환경 (Development Environment)**

* **OS**: Windows / macOS / Linux (Node.js 설치 가능 환경)  
* **Runtime**: Node.js (LTS 버전 권장, v18 이상)  
* **IDE**: VS Code (추천 확장: Tailwind CSS IntelliSense, ES7+ React/Redux/React-Native snippets)  
* **API**: Google AI Studio (Gemini 2.5 Pro / Flash API Key 발급 필요)

## **3\. 핵심 기술 스택 (The Tech Stack)**

* **Frontend**:  
  * **HTML5 / CSS3 (Tailwind CSS)**: 별도 빌드 없이 CDN으로 가볍게 스타일링.  
  * **Vanilla JavaScript**: 복잡한 상태 관리 라이브러리 없이 직접 DOM 제어.  
* **Backend**:  
  * **Express.js**: 초경량 웹 서버 구성.  
  * **LangChain.js**: RAG(검색 증강 생성) 파이프라인 구축의 핵심 엔진.  
* **Data & AI**:  
  * **Google Gemini API**: 텍스트 생성 및 임베딩 담당.  
  * **HNSWLib (Local Vector DB)**: 별도 서버 설치 없이 파일로 저장되는 고성능 벡터 저장소.  
  * **pdf-parse / mammoth**: PDF 및 Word 문서 텍스트 추출용.

## **4\. 환경 셋업 체크리스트 (Setup Checklist)**

1. **Node.js 설치**: node \-v 명령어로 확인.  
2. **프로젝트 초기화**: npm init \-y  
3. **필수 라이브러리 설치**:  
   npm install express @langchain/google-genai @langchain/community hnswlib-node dotenv pdf-parse

4. **.env 파일 생성**: 발급받은 GOOGLE\_API\_KEY를 안전하게 저장.
